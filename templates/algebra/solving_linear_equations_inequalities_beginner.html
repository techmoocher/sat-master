Solving Linear Equations and Inequalities

Master the fundamental skills for solving linear equations on the SAT

Introduction to Linear Equations

Linear equations are the foundation of algebra and appear frequently on the SAT. A linear equation can be written in the form: ax + b = c, where a, b, and c are constants.

Key Concepts
  • Isolating variables by performing operations on both sides of the equation
  • Combining like terms
  • Identifying solutions vs. no solution vs. infinite solutions
  • Multi-step equation solving techniques

Basic Steps for Solving Linear Equations

The basic principle of solving any equation is to perform the same operation on both sides to maintain equality. Keep these steps in mind:

  1. Simplify both sides of the equation by combining like terms
  2. Use addition or subtraction to isolate variable terms on one side
  3. Use multiplication or division to solve for the variable
  4. Check your solution in the original equation

Example 1: Solve for x in 3x + 7 = 4x - 5

Step 1: Group all variable terms on one side and constants on the other.

3x - 4x = -5 - 7

Step 2: Simplify both sides.

-x = -12

Step 3: Solve for the variable.

x = 12

Step 4: Verify your solution.

3(12) + 7 = 4(12) - 5

36 + 7 = 48 - 5

43 = 43 ✓

Special Cases in Linear Equations

The SAT often tests your understanding of these special cases:

No Solution

When you end up with a false statement like 0 = 5, the equation has no solution.

Example: 2x + 3 = 2x - 1

2x - 2x = -1 - 3

0 = -4 (False statement)

Therefore, there is no solution.

Infinite Solutions

When you end up with a true statement like 0 = 0, the equation has infinitely many solutions.

Example: 3(x - 2) = 3x - 6

3x - 6 = 3x - 6

0 = 0 (True statement)

Therefore, all real values of x are solutions.

Linear Inequalities

Linear inequalities are similar to linear equations but use inequality symbols (<, >, ≤, ≥) instead of equals signs. The key difference is that when you multiply or divide both sides by a negative number, you must flip the inequality sign.

Example 2: Solve for x in -2x + 3 > 7

Step 1: Isolate the variable term.

-2x > 7 - 3

-2x > 4

Step 2: Solve for the variable. Since we're dividing by a negative number (-2), we need to flip the inequality sign.

x < -2

Common Mistake: Forgetting to Flip the Sign

One of the most common errors on the SAT is forgetting to flip the inequality sign when multiplying or dividing by a negative number. Always double-check your work when dealing with negatives and inequalities!

Compound Inequalities

The SAT may include compound inequalities, which involve two inequality conditions at once.

Example 3: Solve for x in -3 < 2x + 1 < 7

Step 1: Solve each part of the inequality separately.

-3 < 2x + 1 and 2x + 1 < 7

Step 2: Subtract 1 from all parts.

-4 < 2x and 2x < 6

Step 3: Divide all parts by 2.

-2 < x and x < 3

Step 4: Write as a single inequality.

-2 < x < 3

Practice Questions

1. Solve for x: 5x - 3 = 2x + 9

5x - 3 = 2x + 9
5x - 2x = 9 + 3
3x = 12
x = 4

2. Solve for x: 3(x + 2) - 4 = 2(x - 1) + 5

3(x + 2) - 4 = 2(x - 1) + 5
3x + 6 - 4 = 2x - 2 + 5
3x + 2 = 2x + 3
3x - 2x = 3 - 2
x = 1

3. Solve for the inequality: -4x + 5 ≤ -3

-4x + 5 ≤ -3
-4x ≤ -3 - 5
-4x ≤ -8
x ≥ 2 (Remember to flip the inequality sign when dividing by a negative)

4. Which value of x is a solution to 3(2x - 4) = 6x - 12?

3(2x - 4) = 6x - 12
6x - 12 = 6x - 12
0 = 0
This equation has infinitely many solutions. Any value of x would work.

5. Solve for the compound inequality: -1 ≤ 3x - 5 < 7

-1 ≤ 3x - 5 < 7
-1 + 5 ≤ 3x < 7 + 5
4 ≤ 3x < 12
4/3 ≤ x < 4
So the solution is: 4/3 ≤ x < 4

SAT Tips

  • Always read the problem carefully to understand what variable you are solving for.
  • Show your work neatly to avoid careless errors.
  • Check your answer by substituting it back into the original equation.
  • Watch out for special cases where the solution might be "no solution" or "infinite solutions."
  • Be careful with negative coefficients in inequalities - don't forget to flip the sign!
  • On the SAT, look for shortcuts - sometimes you can test answer choices rather than solving the full equation.
