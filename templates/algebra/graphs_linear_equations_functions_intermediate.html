Graphs of Linear Equations and Functions

Intermediate level: Systems of equations and complex applications

Systems of Linear Equations

Key Concepts
  • Solving systems by graphing
  • Solutions as points of intersection
  • Special cases (parallel and coincident lines)
  • Applications in real-world problems

Graphical Solutions to Systems

When two lines intersect, their point of intersection represents the solution to the system of equations. This is because at that point, both equations are satisfied simultaneously.

Example 1: Solve the system graphically:

y = 2x + 1

y = -x + 4

Step 1: Graph the first line: y = 2x + 1

This has slope m = 2 and y-intercept b = 1

Step 2: Graph the second line: y = -x + 4

This has slope m = -1 and y-intercept b = 4

Step 3: Find the intersection point

The lines appear to intersect at the point (1, 3)

Step 4: Verify the solution algebraically

Substitute (1, 3) into both equations:

Line 1: 3 = 2(1) + 1 = 3 ✓

Line 2: 3 = -1 + 4 = 3 ✓

Therefore, the solution is (1, 3).

Graphical representation of system with intersection at (1, 3)

Example 2: Solve the system using the substitution method:

2x + 3y = 12

y = 4 - x

Step 1: We already have y isolated in the second equation

Step 2: Substitute the expression for y into the first equation

2x + 3(4 - x) = 12

2x + 12 - 3x = 12

-x + 12 = 12

-x = 0

x = 0

Step 3: Find y by substituting x = 0 back into either equation

y = 4 - 0 = 4

Therefore, the solution is (0, 4).

Special Cases in Systems of Equations

Parallel Lines: No Solution

When two lines have the same slope but different y-intercepts, they never intersect.

Example:

y = 2x + 1

y = 2x + 4

Both lines have slope m = 2, but different y-intercepts. This system has no solution.

Coincident Lines: Infinite Solutions

When two lines are actually the same line, every point on the line is a solution.

Example:

y = 2x + 3

4x + 2y = 6

The second equation can be rewritten as y = -2x + 3, which is the same as the first equation. This system has infinitely many solutions.

Linear Inequalities

Linear inequalities represent regions of the coordinate plane instead of just a line.

Graphing linear inequalities:

  1. Graph the boundary line (solid for ≤ or ≥, dashed for < or >)
  2. Test a point not on the line to determine which side to shade
  3. Shade the region that satisfies the inequality

Example: Graph the inequality y > 2x - 1

Step 1: Graph the boundary line y = 2x - 1 as a dashed line (since > doesn't include the line itself)

Step 2: Test a point not on the line, such as (0, 0)

Substitute (0, 0) into the inequality: 0 > 2(0) - 1

0 > -1, which is true

Step 3: Since the test point satisfies the inequality, shade the region containing (0, 0), which is above the line

Systems of Linear Inequalities

To graph a system of linear inequalities:

  1. Graph each inequality separately
  2. Identify the region where all inequalities are satisfied (the intersection of the shaded regions)

Example: Graph the system of inequalities

y ≤ 2x + 1

y ≥ -x + 3

Step 1: Graph y = 2x + 1 as a solid line (since ≤ includes the line)

Step 2: Shade below this line

Step 3: Graph y = -x + 3 as a solid line (since ≥ includes the line)

Step 4: Shade above this line

Step 5: The solution region is where both shadings overlap

Real-World Applications

Example 1: Break-Even Analysis

A company produces widgets with a fixed cost of $5000 and a variable cost of $8 per widget. The widgets sell for $20 each.

Step 1: Define variables

Let x = number of widgets produced and sold

Step 2: Write equations for cost and revenue

Cost (C) = 5000 + 8x

Revenue (R) = 20x

Step 3: Find the break-even point by setting C = R

5000 + 8x = 20x

5000 = 12x

x = 417 (rounded up from 416.67)

Step 4: Interpret the result

The company must sell 417 widgets to break even.

Example 2: Motion Problems

Two cars start at the same point. Car A travels east at 60 mph, while Car B travels north at 45 mph. How far apart are they after 2 hours?

Step 1: Define position functions

Car A: x = 60t, y = 0 (where t is time in hours)

Car B: x = 0, y = 45t

Step 2: Calculate positions at t = 2

Car A at t = 2: (120, 0)

Car B at t = 2: (0, 90)

Step 3: Apply the Pythagorean theorem to find the distance

d = √(120² + 90²) = √(14400 + 8100) = √22500 = 150 miles

Practice Problems

1. A theater charges $8 for adult tickets and $5 for child tickets. If 200 tickets were sold for a total of $1400, how many of each type were sold?

Let x = number of adult tickets

Let y = number of child tickets

Equations:

x + y = 200 (total tickets)

8x + 5y = 1400 (total revenue)

From the first equation:

y = 200 - x

Substitute into the second equation:

8x + 5(200 - x) = 1400

8x + 1000 - 5x = 1400

3x = 400

x = 133

Substitute x = 133 into the first equation:

y = 200 - 133

y = 67

Solution: 133 adult tickets and 67 child tickets

2. A cyclist travels at 15 mph with no wind. With a tailwind, the cyclist can travel 22.5 mph, and against the same wind, the cyclist can travel 10 mph. What is the speed of the wind?

Let c = cyclist's speed with no wind

Let w = wind speed

With tailwind: c + w = 22.5

Against the wind: c - w = 10

Add the two equations:

(c + w) + (c - w) = 22.5 + 10

2c = 32.5

c = 16.25

Substitute c = 16.25 into the first equation:

16.25 + w = 22.5

w = 6.25

Solution: The wind speed is 6.25 mph.

3. Graph the solution region for the system of inequalities:

y ≤ 2x + 3

y ≥ -x + 1

x ≥ 0

Step 1: Graph each inequality

- y ≤ 2x + 3: Graph y = 2x + 3 as a solid line and shade below

- y ≥ -x + 1: Graph y = -x + 1 as a solid line and shade above

- x ≥ 0: Graph x = 0 (the y-axis) as a solid line and shade to the right

Step 2: Find the region that satisfies all three inequalities

This creates a triangular region bounded by:

- The y-axis (x = 0)

- The line y = -x + 1

- The line y = 2x + 3

Step 3: Find the vertices of this region

At the y-axis, set x = 0 in both lines:

For y = -x + 1: When x = 0, y = 1

For y = 2x + 3: When x = 0, y = 3

So two vertices are (0, 1) and (0, 3)

For the third vertex, find the intersection of the two lines:

-x + 1 = 2x + 3

-3x = 2

x = -2/3

Since x = -2/3 is negative, it violates the constraint x ≥ 0.

Solution: The vertices of the triangular region are (0, 1), (0, 3), and the intersection of y = -x + 1 and y = 2x + 3 within the feasible region.
