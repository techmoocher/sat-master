Linear Programming and Optimization

Key Concepts
  • Setting up optimization problems with linear constraints
  • Finding feasible regions using systems of inequalities
  • Identifying optimal values at vertices
  • Interpreting solutions in context

Introduction to Linear Programming

Linear programming is a powerful mathematical technique used to find the best outcome in a mathematical model whose requirements are represented by linear relationships. It's widely used in business, economics, and engineering to optimize processes and allocate resources.

Components of a Linear Programming Problem:

  1. Objective Function: The quantity we want to maximize or minimize (e.g., profit, cost)
  2. Decision Variables: The quantities we need to determine (usually represented as x, y, etc.)
  3. Constraints: Limitations expressed as linear inequalities
  4. Non-negativity Conditions: Decision variables are typically restricted to be non-negative

Graphical Method of Linear Programming

Example 1: A company manufactures two products: A and B. Each unit of product A requires 2 hours of machine time and 1 hour of labor. Each unit of product B requires 1 hour of machine time and 3 hours of labor. The company has 40 hours of machine time and 30 hours of labor available per week. If the profit is $50 per unit for product A and $40 per unit for product B, how many units of each product should the company produce to maximize profit?

Step 1: Define the decision variables

Let x = number of units of product A produced per week

Let y = number of units of product B produced per week

Step 2: Write the objective function

Maximize P = 50x + 40y

Step 3: Identify the constraints

Machine time constraint: 2x + y ≤ 40

Labor constraint: x + 3y ≤ 30

Non-negativity constraints: x ≥ 0, y ≥ 0

Step 4: Graph the feasible region

The feasible region is the shaded area that satisfies all constraints:

  • Draw the line 2x + y = 40
  • Draw the line x + 3y = 30
  • Shade the region below both lines in the first quadrant (where x ≥ 0 and y ≥ 0)

Step 5: Find the vertices of the feasible region

The vertices are:

  • Origin: (0, 0)
  • x-axis intercept of machine constraint: (20, 0)
  • y-axis intercept of machine constraint: (0, 40)
  • x-axis intercept of labor constraint: (30, 0)
  • y-axis intercept of labor constraint: (0, 10)
  • Intersection of the two constraint lines:

To find the intersection, solve the system:

2x + y = 40

x + 3y = 30

From the first equation: y = 40 - 2x

Substituting into the second equation:

x + 3(40 - 2x) = 30

x + 120 - 6x = 30

-5x = -90

x = 18

Substituting back: y = 40 - 2(18) = 40 - 36 = 4

So the vertices of the feasible region are (0, 0), (20, 0), (18, 4), and (0, 10)

Step 6: Evaluate the objective function at each vertex

P(0, 0) = 50(0) + 40(0) = 0

P(20, 0) = 50(20) + 40(0) = 1000

P(18, 4) = 50(18) + 40(4) = 900 + 160 = 1060

P(0, 10) = 50(0) + 40(10) = 400

Step 7: Identify the optimal solution

The maximum profit is $1060, achieved by producing 18 units of product A and 4 units of product B per week.

Corner Point Principle

A fundamental principle in linear programming is that if an optimal solution exists, it will occur at a corner point (vertex) of the feasible region. This is because a linear function increases or decreases most rapidly along the edges of the feasible region.

Why this works: The objective function represents a family of parallel lines (in 2D) or planes (in higher dimensions). As we move these lines/planes in the direction of increasing the objective value, the last point of contact with the feasible region will be a vertex.

Special Cases in Linear Programming

1. Multiple Optimal Solutions

When the objective function line is parallel to one of the constraint boundaries, there might be infinitely many optimal solutions along that boundary.

Example:

Maximize P = x + y

Subject to:

x + y ≤ 8

x ≥ 0, y ≥ 0

Here, any point on the line segment from (8, 0) to (0, 8) gives the same optimal value P = 8.

2. Unbounded Solution

If the feasible region extends infinitely in the direction of increasing objective function, there is no maximum value.

Example:

Maximize P = x + y

Subject to:

x - y ≤ 4

x ≥ 0, y ≥ 0

No matter how large P becomes, we can always find points in the feasible region with even larger values.

3. Infeasible Solution

When there are no points that satisfy all constraints simultaneously, there is no feasible solution.

Example:

Subject to:

x + y ≤ 5

x + y ≥ 10

x ≥ 0, y ≥ 0

These constraints are contradictory, so no solution exists.

4. Degenerate Solution

Occurs when more than two constraints intersect at a vertex, potentially causing computational complications in algorithms.

On the SAT, you'll primarily be concerned with cases 1-3 rather than degenerate solutions.

Advanced Applications

Example 2: Resource Allocation Problem

A factory produces two types of bicycles: mountain bikes and road bikes. Each mountain bike requires 3 hours of assembly and 2 hours of painting. Each road bike requires 2 hours of assembly and 1 hour of painting. The factory has 60 hours of assembly time and 40 hours of painting time available per week. Market research indicates that at most 15 mountain bikes and at most 20 road bikes can be sold per week. If the profit is $80 per mountain bike and $60 per road bike, how many of each should be produced to maximize profit?

Step 1: Define the decision variables

Let x = number of mountain bikes produced per week

Let y = number of road bikes produced per week

Step 2: Write the objective function

Maximize P = 80x + 60y

Step 3: Identify the constraints

Assembly time: 3x + 2y ≤ 60

Painting time: 2x + y ≤ 40

Market constraints: x ≤ 15, y ≤ 20

Non-negativity: x ≥ 0, y ≥ 0

Step 4: Graph the feasible region

The feasible region is bounded by all five constraints.

Step 5: Find the vertices of the feasible region

The potential vertices include:

  • (0, 0): Origin
  • (0, 20): Maximum road bikes, no mountain bikes
  • (15, 0): Maximum mountain bikes, no road bikes
  • Intersection of x = 15 and y = 20: (15, 20)
  • Intersection of assembly constraint and y = 20: Solving 3x + 2(20) ≤ 60, we get 3x + 40 ≤ 60, so x ≤ 6.67. This gives (6.67, 20)
  • Intersection of painting constraint and y = 20: Solving 2x + 20 ≤ 40, we get x ≤ 10. This gives (10, 20)
  • Intersection of x = 15 and assembly constraint: Solving 3(15) + 2y ≤ 60, we get 45 + 2y ≤ 60, so y ≤ 7.5. This gives (15, 7.5)
  • Intersection of x = 15 and painting constraint: Solving 2(15) + y ≤ 40, we get 30 + y ≤ 40, so y ≤ 10. This gives (15, 10)
  • Intersection of assembly and painting constraints: From assembly, y = (60 - 3x)/2. From painting, y = 40 - 2x. Setting these equal: (60 - 3x)/2 = 40 - 2x, which gives x = 8, y = 24. But since y > 20, this point is outside our feasible region.

The feasible vertices are (0, 0), (0, 20), (10, 20), (15, 10), (15, 7.5), (15, 0).

Step 6: Evaluate the objective function at each vertex

P(0, 0) = 0

P(0, 20) = 60(20) = 1200

P(10, 20) = 80(10) + 60(20) = 800 + 1200 = 2000

P(15, 10) = 80(15) + 60(10) = 1200 + 600 = 1800

P(15, 7.5) = 80(15) + 60(7.5) = 1200 + 450 = 1650

P(15, 0) = 80(15) = 1200

Step 7: Identify the optimal solution

The maximum profit is $2000, achieved by producing 10 mountain bikes and 20 road bikes per week.

Example 3: Diet Problem

A nutritionist is planning a diet using two types of food. Food A contains 2 units of protein, 1 unit of calcium, and 3 units of iron per serving. Food B contains 1 unit of protein, 2 units of calcium, and 1 unit of iron per serving. The diet must include at least 10 units of protein, 7 units of calcium, and 12 units of iron. If Food A costs $2 per serving and Food B costs $3 per serving, how many servings of each food should be included to minimize cost?

Step 1: Define the decision variables

Let x = number of servings of Food A

Let y = number of servings of Food B

Step 2: Write the objective function (to minimize)

Minimize C = 2x + 3y

Step 3: Identify the constraints

Protein: 2x + y ≥ 10

Calcium: x + 2y ≥ 7

Iron: 3x + y ≥ 12

Non-negativity: x ≥ 0, y ≥ 0

Step 4: Find the vertices of the feasible region

We need to find the intersection points of:

  • Protein and Calcium: 2x + y = 10 and x + 2y = 7
  • Protein and Iron: 2x + y = 10 and 3x + y = 12
  • Calcium and Iron: x + 2y = 7 and 3x + y = 12

For Protein and Calcium:

From Protein: y = 10 - 2x

Substituting into Calcium: x + 2(10 - 2x) = 7

x + 20 - 4x = 7

-3x = -13

x = 13/3 ≈ 4.33

y = 10 - 2(13/3) = 10 - 26/3 = 30/3 - 26/3 = 4/3 ≈ 1.33

For Protein and Iron:

2x + y = 10

3x + y = 12

Subtracting: -x = -2

x = 2

y = 10 - 2(2) = 6

For Calcium and Iron:

x + 2y = 7

3x + y = 12

Multiply first equation by 3: 3x + 6y = 21

Subtract second: 5y = 9

y = 9/5 = 1.8

x = 7 - 2(9/5) = 7 - 18/5 = 35/5 - 18/5 = 17/5 = 3.4

So the vertices are approximately: (4.33, 1.33), (2, 6), and (3.4, 1.8)

Step 5: Evaluate the objective function at each vertex

C(4.33, 1.33) = 2(4.33) + 3(1.33) = 8.66 + 3.99 = 12.65

C(2, 6) = 2(2) + 3(6) = 4 + 18 = 22

C(3.4, 1.8) = 2(3.4) + 3(1.8) = 6.8 + 5.4 = 12.2

Step 6: Identify the optimal solution

The minimum cost is $12.20, achieved by including 3.4 servings of Food A and 1.8 servings of Food B.

Since it's impractical to serve partial servings, we would round to 3 servings of Food A and 2 servings of Food B, with a cost of $12.

Advanced Practice Problems

1. A company manufactures two types of products: standard and deluxe. Each standard product requires 3 hours of labor and 4 hours of machine time. Each deluxe product requires 5 hours of labor and 3 hours of machine time. The company has 60 hours of labor and 48 hours of machine time available per day. The profit is $10 per standard product and $15 per deluxe product. How many of each type should be produced to maximize profit?

Step 1: Define variables

Let x = number of standard products

Let y = number of deluxe products

Step 2: Write the objective function

Maximize P = 10x + 15y

Step 3: Identify constraints

Labor: 3x + 5y ≤ 60

Machine: 4x + 3y ≤ 48

Non-negativity: x ≥ 0, y ≥ 0

Step 4: Find vertices

(0, 0), (0, 12), (12, 0)

Find the intersection of constraints:

3x + 5y = 60

4x + 3y = 48

Multiply the first by 4: 12x + 20y = 240

Multiply the second by 3: 12x + 9y = 144

Subtract: 11y = 96

y = 8.73

Substitute into first: 3x + 5(8.73) = 60

3x + 43.65 = 60

3x = 16.35

x = 5.45

Step 5: Evaluate objective function

P(0, 0) = 0

P(0, 12) = 15(12) = 180

P(12, 0) = 10(12) = 120

P(5.45, 8.73) = 10(5.45) + 15(8.73) = 54.5 + 130.95 = 185.45

Step 6: Identify optimal solution

The maximum profit is $185.45, achieved by producing 5 standard products and 9 deluxe products (rounding to nearest whole units).

2. A furniture maker is crafting tables and chairs. Each table requires 9 hours of woodworking and 2 hours of finishing. Each chair requires 5 hours of woodworking and 1 hour of finishing. The workshop has 90 hours of woodworking time and 16 hours of finishing time available per week. Due to space limitations, at most 12 tables can be made per week. If each table sells for $120 and each chair for $75, how many of each should be made to maximize revenue?

Step 1: Define variables

Let x = number of tables

Let y = number of chairs

Step 2: Write the objective function

Maximize R = 120x + 75y

Step 3: Identify constraints

Woodworking: 9x + 5y ≤ 90

Finishing: 2x + y ≤ 16

Space: x ≤ 12

Non-negativity: x ≥ 0, y ≥ 0

Step 4: Find vertices

The vertices include (0, 0), (10, 0), (0, 18), (0, 16)

Find intersections:

- Woodworking and Finishing:

From finishing: y = 16 - 2x

Substitute into woodworking: 9x + 5(16 - 2x) = 90

9x + 80 - 10x = 90

-x = 10

x = 10

y = 16 - 2(10) = -4

Since y < 0, this point is outside our feasible region.

- Space and Woodworking:

When x = 12, from woodworking: 9(12) + 5y ≤ 90

108 + 5y ≤ 90

5y ≤ -18

y ≤ -3.6

Since y < 0, this violates non-negativity.

- Space and Finishing:

When x = 12, from finishing: 2(12) + y ≤ 16

24 + y ≤ 16

y ≤ -8

This violates non-negativity.

So the feasible region is bounded by x = 0, y = 0, woodworking, and finishing constraints.

The proper intersection of woodworking and finishing is:

9x + 5y = 90

2x + y = 16

From second equation: y = 16 - 2x

Substitute: 9x + 5(16 - 2x) = 90

9x + 80 - 10x = 90

-x = 10

This gives x = -10, which violates non-negativity.

Let's try a different approach. From finishing: y = 16 - 2x

Looking at points along this line:

When x = 0, y = 16

When x = 8, y = 0

Let's check if these satisfy woodworking:

At (0, 16): 9(0) + 5(16) = 80 ≤ 90 ✓

At (8, 0): 9(8) + 5(0) = 72 ≤ 90 ✓

So the vertices are (0, 0), (8, 0), (0, 16), and perhaps a point where the woodworking constraint intersects either axis.

For woodworking at y = 0: 9x = 90, so x = 10

For woodworking at x = 0: 5y = 90, so y = 18

So the vertices are (0, 0), (10, 0), (8, 0), (0, 18), and (0, 16)

Step 5: Evaluate objective function

R(0, 0) = 0

R(10, 0) = 120(10) = 1200

R(8, 0) = 120(8) = 960

R(0, 18) = 75(18) = 1350

R(0, 16) = 75(16) = 1200

Step 6: Identify optimal solution

The maximum revenue is $1350, achieved by making 0 tables and 18 chairs per week.

3. A shipping company needs to transport goods using two types of containers: standard and refrigerated. Each standard container requires 6 square meters of deck space and can hold 10 tons of cargo. Each refrigerated container requires 8 square meters of deck space and can hold 8 tons of cargo. The ship has 120 square meters of deck space and must transport at least 140 tons of cargo. If a standard container costs $800 to ship and a refrigerated container costs $1000, what is the minimum shipping cost?

Step 1: Define variables

Let x = number of standard containers

Let y = number of refrigerated containers

Step 2: Write the objective function

Minimize C = 800x + 1000y

Step 3: Identify constraints

Deck space: 6x + 8y ≤ 120

Cargo: 10x + 8y ≥ 140

Non-negativity: x ≥ 0, y ≥ 0

Step 4: Find vertices

Find the intersection of the two main constraints:

6x + 8y = 120 (deck space)

10x + 8y = 140 (cargo)

Subtract the first from the second:

4x = 20

x = 5

Substitute back: 6(5) + 8y = 120

30 + 8y = 120

8y = 90

y = 11.25

So one vertex is (5, 11.25)

For deck space at y = 0: 6x = 120, so x = 20

For cargo at y = 0: 10x = 140, so x = 14

For deck space at x = 0: 8y = 120, so y = 15

For cargo at x = 0: 8y = 140, so y = 17.5

Check if (20, 0) satisfies cargo constraint:

10(20) + 8(0) = 200 ≥ 140 ✓

Check if (0, 17.5) satisfies deck space constraint:

6(0) + 8(17.5) = 140 > 120 ✗

So the vertices are (5, 11.25), (14, 0), and (20, 0).

Step 5: Evaluate objective function

C(5, 11.25) = 800(5) + 1000(11.25) = 4000 + 11250 = 15250

C(14, 0) = 800(14) = 11200

C(20, 0) = 800(20) = 16000

Step 6: Identify optimal solution

The minimum cost is $11,200, achieved by shipping 14 standard containers and 0 refrigerated containers.

Note: Since containers come in whole units, we would use 14 standard containers.

SAT Advanced Tips for Linear Programming

  • Always double-check your constraints to ensure they accurately represent the problem conditions.
  • When finding vertices, systematically check the intersections of each pair of constraints.
  • For minimization problems, the objective function should point in the direction of decreasing values.
  • Remember that real-world problems usually require non-negative solutions, and often whole-number (integer) solutions.
  • On the SAT, be prepared to interpret the meaning of your solution in context of the original problem.
  • If your solution involves fractions or decimals in a context where whole numbers are needed, consider rounding appropriately and verifying that the constraints are still satisfied.
