Intermediate Linear Equations and Inequalities

Advanced techniques for tackling more complex SAT linear equation problems

Solving Complex Linear Equations

At the intermediate level, the SAT will test your ability to solve more complex linear equations that involve multiple steps, fractions, decimals, and strategic approaches.

Key Intermediate Skills
  • Clearing fractions and decimals in equations
  • Complex multi-step equations with distribution
  • Variables on both sides with multiple grouping symbols
  • Identifying the most efficient solving strategy
  • Testing potential solutions in complex cases

Equations with Fractions and Decimals

When facing equations with fractions or decimals, multiply all terms by the LCD (least common denominator) or an appropriate power of 10 to eliminate them:

Example 1: Solve for x in $\frac{2x}{3} - \frac{1}{6} = \frac{x+2}{4}$

Step 1: Find the LCD of all denominators (3, 6, and 4) which is 12.

Step 2: Multiply all terms by 12.

12 × $\frac{2x}{3} - 12 × \frac{1}{6} = 12 × \frac{x+2}{4}$

8x - 2 = 3(x+2)

8x - 2 = 3x + 6

Step 3: Solve for x.

8x - 3x = 6 + 2

5x = 8

x = $\frac{8}{5}$ or 1.6

Strategic Solving Techniques

For more complex equations, choosing the right strategy can save time on the SAT:

Substitution Strategy

When given multiple answer choices, try substituting each value to find which one works.

Example: Which of the following is a solution to $2(3x-1) - 4x = x + 7$?

A) x = -2

B) x = -1

C) x = 2

D) x = 3

Rather than solving algebraically, we can test each option.

For x = 3: $2(3(3)-1) - 4(3) = 3 + 7$

$2(9-1) - 12 = 10$

$2(8) - 12 = 10$

$16 - 12 = 10$

$4 = 10$ (false)

Continuing to check options, we would find that C) x = 2 is correct.

Systems of Linear Equations

The SAT may include systems of linear equations that require you to find values satisfying multiple equations simultaneously.

Example 2: Solve the system of equations:

$3x + 2y = 7$

$4x - y = 5$

Method 1: Elimination

Multiply the second equation by 2:

$3x + 2y = 7$

$8x - 2y = 10$

Add the equations:

$11x = 17$

$x = \frac{17}{11}$

Substitute back to find y:

$3(\frac{17}{11}) + 2y = 7$

$\frac{51}{11} + 2y = 7$

$2y = 7 - \frac{51}{11}$

$2y = \frac{77-51}{11}$

$2y = \frac{26}{11}$

$y = \frac{13}{11}$

Solution: $x = \frac{17}{11}$, $y = \frac{13}{11}$

Advanced Linear Inequalities

Intermediate inequalities often involve multiple variables, absolute values, or compound conditions:

Example 3: Solve the inequality: $|2x - 3| \leq 5$

Step 1: An absolute value inequality like $|ax + b| \leq c$ (where c > 0) can be rewritten as:

$-c \leq ax + b \leq c$

Step 2: Applying to our inequality:

$-5 \leq 2x - 3 \leq 5$

Step 3: Solve for x by adding 3 to all parts:

$-2 \leq 2x \leq 8$

Step 4: Divide all parts by 2:

$-1 \leq x \leq 4$

Step 5: The solution is all values of x between -1 and 4, inclusive.

Common SAT Trap

The SAT frequently includes inequalities where you need to analyze what happens when the variable approaches the boundaries or "edge cases." Always check if the boundaries themselves are included in your solution.

Applying Algebraic Properties

Using algebraic properties allows for more elegant solutions:

Example 4: If $a + b = 10$ and $ab = 21$, find $a^2 + b^2$.

Key insight: We can use the algebraic identity: $(a + b)^2 = a^2 + 2ab + b^2$

Step 1: Rearrange to isolate $a^2 + b^2$

$a^2 + b^2 = (a + b)^2 - 2ab$

Step 2: Substitute the given values

$a^2 + b^2 = 10^2 - 2(21)$

$a^2 + b^2 = 100 - 42$

$a^2 + b^2 = 58$

Practice Questions

1. Solve for x: $\frac{x+2}{3} - \frac{x-1}{4} = \frac{5}{6}$

Find LCD of denominators (3, 4, 6) which is 12.
Multiply all terms by 12:
$4(x+2) - 3(x-1) = 10$
$4x + 8 - 3x + 3 = 10$
$x + 11 = 10$
$x = -1$

2. Solve the system of equations:
$2x + 5y = 16$
$4x - 3y = 2$

Multiply first equation by 2:
$4x + 10y = 32$
$4x - 3y = 2$
Subtract the second equation from the first:
$13y = 30$
$y = \frac{30}{13}$
Substitute back:
$2x + 5(\frac{30}{13}) = 16$
$2x + \frac{150}{13} = 16$
$2x = 16 - \frac{150}{13}$
$2x = \frac{208 - 150}{13}$
$2x = \frac{58}{13}$
$x = \frac{29}{13}$
Solution: $x = \frac{29}{13}$, $y = \frac{30}{13}$

3. Solve the inequality: $\frac{3x+1}{2} > \frac{x-4}{3}$

Multiply both sides by 6 (LCD of 2 and 3):
$3(3x+1) > 2(x-4)$
$9x+3 > 2x-8$
$9x-2x > -8-3$
$7x > -11$
$x > -\frac{11}{7}$

4. For what values of k is the following equation true for all real values of x?
$k(3x - 1) = 3kx - k$

Expand the left side:
$3kx - k = 3kx - k$
This is an identity (true for all values of x) regardless of what k equals.
Therefore, the equation is true for all values of k.

5. Solve the inequality: $|3x - 2| + 1 \geq 7$

$|3x - 2| + 1 \geq 7$
$|3x - 2| \geq 6$
This means either:
$3x - 2 \geq 6$ or $3x - 2 \leq -6$
Solving the first inequality:
$3x \geq 8$
$x \geq \frac{8}{3}$
Solving the second inequality:
$3x \leq -4$
$x \leq -\frac{4}{3}$
The solution is: $x \leq -\frac{4}{3}$ or $x \geq \frac{8}{3}$

SAT Intermediate Strategies

  • When dealing with fractions, always multiply all terms by the LCD to clear denominators.
  • For systems of equations, decide whether substitution or elimination is more efficient based on the coefficients.
  • In absolute value problems, remember to consider both possible cases (positive and negative).
  • Identify algebraic identities like $(a+b)^2 = a^2 + 2ab + b^2$ to solve certain problems more quickly.
  • For complex problems, sometimes testing answer choices is faster than solving directly.
  • Always double-check your final answer by substituting back into the original equation.
